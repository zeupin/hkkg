# hkkg

通用边栏菜单。

![](demo1.png)

## 层次结构

```
.hkkg                       主类
    .hkkg-caption           大标题
    .hkkg-box1              一级box容器
        .hkkg-box2          二级box容器
            .hkkg-box3      三级box容器
        .hkkg-t1            容器内的一级标题
        .hkkg-t2            容器内的二级标题
        .hkkg-t3            容器内的三级标题

    .hkkg-box1.shrink       收起状态的box
    .hkkg-box1.shrinkable   可折叠的box
```
